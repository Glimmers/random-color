## random-color

Module to generate random RGB colors

```js
var randomColor = require('random-color')

randomColor()
// => rgb(230, 120, 50)

// Set maximum
randomColor(25)
// => rgb(22, 13, 5)

// Set maximum, minimum 
randomColor(50, 25)
// => rgb(32, 40, 26)
```

## Install

```bash
$ npm install random-color
```
